Lista de Definiciones

HTML
Lenguaje de marcado utilizado para estructurar páginas web.
CSS
Lenguaje de hojas de estilo utilizado para dar formato a la presentación de documentos HTML.
JavaScript
Lenguaje de programación que se utiliza para crear contenido dinámico en las páginas web.
HTTP
Protocolo de transferencia de hipertexto, utilizado para la comunicación entre clientes y servidores web.
URL
Dirección web que especifica la ubicación de un recurso en la web.
